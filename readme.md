# 功能
- 操纵字幕文件, 实现字幕的平移, 以和视频匹配. 
    有时候下载好了视频和字幕, 却发现找到的字幕和视频没有对齐, 而是有几秒的不同步, 于是写了一个小脚本来处理不同步的字幕.  
- 我测试了*.srt *.ass都可以用.    
- 已打包.exe文件, 不需要安装python环境.
